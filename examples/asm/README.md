# Assembly Example

This is a simple RISC-V assembly program that prints a string to the 8051 UART.


## Prerequisites

* [SDCC][sdcc]
* [GNU Binutils][binutils] for riscv32-elf


## Usage

Run `make sim` in this directory to build the example and start the simulator.
